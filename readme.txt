* Android application to fetch the Yahoo weather information by using Yahoo! Geoplanet and Yahoo! Weather RSS feed API's.

* Used Android SDK and Eclipse editor to develop this application and Facebook Connect API to share the weather information on Facebook.

* User needs to plug in either the zip code or city name followed by the state name in the search box to fetch the weather information
pertaining to that city or zip code.
